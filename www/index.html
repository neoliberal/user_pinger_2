Manage Pings

You can manage your subscriptions on this page.

You need mod approval to subscribe to a protected group. Message the mods to request access.

  • ✋ Protected: only mods can subscribe
  • 🔒 Locked: only mods can ping
  • 🙈 Hidden: group does not appear in documentation (except to mods)
  • 📩 Activity: how many pings the group has received in the last 7 days

Mod Functions

No need to delete groups, just set locked, protected, and hidden. Use the "subscribe user" boxes above to subscribe either yourself or someone else to a protected group.
